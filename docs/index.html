盯盘助手隐私政策

盯盘助手是一款用于监控指数和股票价格，并在价格到达或穿过用户设定目标价时发出浏览器提醒的 Chrome 扩展。

1. 扩展的单一用途

盯盘助手的单一用途是在浏览器中帮助用户查看行情、设置目标价提醒，并在价格触发时通过通知和扩展角标提醒用户。

2. 我们会处理哪些数据

  • 用户主动设置的关注股票列表
  • 用户主动设置的目标价提醒
  • 本地缓存的最近一次行情数据和提醒状态

以上数据主要保存在用户浏览器本地存储中，不用于广告、画像或账号识别。

3. 权限用途说明

  • storage：保存关注列表、目标价和本地提醒状态
  • notifications：在价格触发时发送浏览器通知
  • alarms：定时轮询行情，保证在弹窗关闭时仍能监控价格
  • searchapi.eastmoney.com：搜索证券代码和名称
  • push2.eastmoney.com：获取指数和股票行情

4. 数据共享与传输

盯盘助手不会将用户关注列表、目标价提醒或本地状态上传到开发者自建服务器。扩展仅会访问第三方公开行情接口，以获取证券搜索和市场行情数据。

5. 联系方式

如果你对本隐私政策有任何问题，请通过你在 Chrome Web Store 开发者资料中公开的联系邮箱与开发者联系。
